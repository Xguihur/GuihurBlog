2. 主题追求极简，根据 vuepress 的默认主题修改而成，官方的主题配置仍然适用；<br>
3. 欢迎大家来到我的网站，希望你能喜欢！
:::

## Use
::: tip 项目启动及打包
**Build**

```bash
npm run build
```

**Server**

```bash
npm run dev
```

**Deploy**

```bash
npm run dev
```
:::

## 里程碑

::: warning 站点建立
