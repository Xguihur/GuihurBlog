# 写作模板索引（内部）

这些模板仅供仓库内部写作时复制使用，不对外展示。

## 博客模板

1. `blog-life.md`
2. `blog-thoughts.md`
3. `blog-sports.md`
4. `blog-meta.md`

## 文档模板

1. `doc-tech.md`
2. `doc-sports.md`

## 使用建议

1. 新建文章时，先复制对应模板到目标目录，再改 `title/date/tags`。
2. `categories` 保持模板给定值，不要混用旧分类名。
3. 博客路径建议：`blogs/<category>/<year>/<mmddNN>.md`。
